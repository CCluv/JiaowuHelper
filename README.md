C#正方教务助手
=================================================

正方教务系统的学生助手，模拟执行一些简易操作，乐趣多多。


## 使用说明

* 填写对应的教务系统地址，刷新验证码登录即可

## 运行要求

* .Net 3.5 及以上环境即可
